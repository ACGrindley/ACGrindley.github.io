Installation

1. Download git from https://git-scm.com/download/win (2.45.1 on 21/05/2024) 2. Install with default options. 3. Navigate to project folder, right click, "Open Git Bash here". 4. Initialise Git repository at location "git init". 5. Check files in current directory "ls".

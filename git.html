Installation

1. Download git from https://git-scm.com/download/win (2.45.1 on 21/05/2024)
2. Install with default options.
3. Create a username "git config --global user.name "ACGrindley"" and email "git config --global user.email "ashleygrindley@live.co.uk"".
4. Navigate to project folder, right click, "Open Git Bash here".
5. Initialise Git repository at location "git init".
6. Check files in current directory "ls".
7. Check status of files in current directory "git status".
8. Stage a file "git add filename.filetype".
9. Stage all files in current directory "git add --all".
10. Commit the staged files "git commit -m "Text to include in commit message"".
11. Check the log to see history of commits "git log".
12. Add remote repository with URL as origin to local Git repo on GitHub "git remote add origin https://github.com/ACGrindley/my_website.git".
13. Push master branch to the origin URL and set as default remote branch "git push --set-upstream origin master".
